## 2. 完整规划 Pipeline（8 步）

```
   用户输入
      │
      ▼
┌──────────────────────────────────────────────────────────────┐
│ Step 1  城市提取        _extract_city()  regex 匹配 300+ 城市  │
└──────────────────────────────────────────────────────────────┘
      │
      ▼
┌──────────────────────────────────────────────────────────────┐
│ Step 2  意图解析        Intent Agent → LLM 调用                │
│          输出: origin / destination / keywords / UserProfile  │
└──────────────────────────────────────────────────────────────┘
      │
      ▼
┌──────────────────────────────────────────────────────────────┐
│ Step 3  地理编码        robust_geocode()  ×2（起点 + 终点）     │
│           4 层兜底: geocode → input_tips → 补后缀 → 正则回退    │
└──────────────────────────────────────────────────────────────┘
      │
      ▼
┌──────────────────────────────────────────────────────────────┐
│ Step 4  POI 获取                                               │
│                                                                │
│   ┌─ DB 路径 (USE_POI_DB=true) ──────────┐                    │
│   │  预计算聚类召回 → 推荐引擎排序        │                    │
│   └──────────────────────────────────────┘                    │
│                   或                                           │
│   ┌─ API 路径 (默认) ────────────────────┐                    │
│   │  POI Strategy Agent (LLM) → 高德搜索  │                    │
│   │  → evaluate_pois (LLM) → 自动补搜    │                    │
│   └──────────────────────────────────────┘                    │
└──────────────────────────────────────────────────────────────┘
      │
      ▼
┌──────────────────────────────────────────────────────────────┐
│ Step 5  路线算法        Route Engine                           │
│                                                                │
│   build_graph()                     shortest_path()           │
│   ┌──────────────────┐              ┌──────────────────┐      │
│   │ 起点 + POI + 终点 │──────────────│ 投影到起终点连线  │      │
│   │ 全连接邻接矩阵    │              │ 均分 N 段         │      │

│   └──────────────────┘              │ 自动减站          │      │
│                                     └──────────────────┘      │
└──────────────────────────────────────────────────────────────┘
      │
      ▼
┌──────────────────────────────────────────────────────────────┐
│ Step 6  解说              Narrator Agent → LLM 个性化解说       │
└──────────────────────────────────────────────────────────────┘
      │
      ▼
┌──────────────────────────────────────────────────────────────┐
│ Step 7  审核              Reviewer Agent → LLM 质量审计        │
│                         最多 2 轮 refine                       │
│                         不通过 ──→ 补搜 → 重规划 → 再审核       │
└──────────────────────────────────────────────────────────────┘
      │
      ▼
┌──────────────────────────────────────────────────────────────┐
│ Step 8  输出                                                   │
│                                                                │
```

## 5. Web 请求链路

```
   Browser
      │
      │  POST /api/plan/stream  {"query": "从西安北站出发去曲江玩"}
      ▼
┌─────────────────────────────────────────────┐
│  FastAPI  web/server.py                     │
│                                             │
│  ① 创建 asyncio.Queue  +  session_id        │
│  ② 注册 _on_progress 回调                    │
│  ③ ThreadPoolExecutor 提交 run_multi_agent  │
│  ④ SSE 事件循环: 边等结果边推进度             │
│  ⑤ 结果到达 → _build_response() → SSE 推送   │
└─────────────────────────────────────────────┘
      │                              ▲
      │  SSE stream                  │ queue.put_nowait()
      ▼                              │
┌─────────────────────────────────────────────┐
│  Browser  逐条接收 SSE 事件                  │
│                                             │
│  data: {"type":"progress","emoji":"📍","msg":"城市：西安"}   │
│  data: {"type":"progress","emoji":"🧠","msg":"Intent..."}   │
│  ...                                        │
│  data: {"type":"result","narration":"...","mermaid":"...","stops":[...]}
│                                             │
│  渲染: renderMarkdown() + renderMermaid() + renderMap()  │
└─────────────────────────────────────────────┘
```
